# @horde/hd

Horde CLI - multi-agent workspace manager for coordinating AI coding agents.

## Installation

```bash
npm install -g @horde/hd
```

This will download the appropriate native binary for your platform during installation.

## Usage

```bash
# Check version
hd version

# Initialize a new encampment
hd init

# View status
hd status

# List warbands
hd warband list
```

## Supported Platforms

- macOS (Intel and Apple Silicon)
- Linux (x64 and ARM64)
- Windows (x64)

## Manual Installation

If npm installation fails, you can download binaries directly from:
https://github.com/OWNER/horde/releases
